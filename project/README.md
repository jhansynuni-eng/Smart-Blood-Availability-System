# Smart Blood Availability & Delivery Ecosystem

An AI-powered, real-time smart platform connecting blood donors, hospitals, and blood banks to ensure quick and efficient blood availability during emergencies. 

## 🚀 Features

- **Real-Time Inventory Scanning:** Instantly checks nearby blood banks for required blood groups.
- **Smart Donor Auto-Alert System:** Automatically alerts nearby eligible donors via App and SMS when blood is out of stock.
- **Doorstep Delivery Assignment:** AI assigns the fastest delivery mode (Emergency Drone, Fast Urban Bike, or Ambulance) based on urgency and distance.
- **Verified Donor Registration:** Secure registration system to keep track of donor health and eligibility.

## 🛠 Tech Stack

- HTML5
- Vanilla CSS3 (Glassmorphism & Premium UI)
- Vanilla JavaScript (DOM manipulation & logic simulation)

## 🌐 How to Deploy (Vercel)

This project requires zero build steps and is ready to deploy directly.
1. Push this repository to GitHub.
2. Go to [Vercel](https://vercel.com/) and click **Add New Project**.
3. Import your GitHub repository.
4. Click **Deploy** (No framework preset required).

## 💻 Local Development

To run the project locally, simply start a local web server in the root directory:
```bash
python -m http.server 8080
```
Then navigate to `http://localhost:8080` in your browser.
